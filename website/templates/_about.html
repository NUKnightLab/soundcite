About SoundCite

SoundCite is one of a series of tools for content creators produced by the Northwestern University Knight Lab. It was created by Medill student Tyler Fisher and Medill Asst. Professor Jeremy Gilbert.

This project's source code is hosted on GitHub. We encourage you to contribute to the project and we value your feedback. Feel free to report bugs and discuss features on the GitHub issues page. You can also send email to knightlab@northwestern.edu.
